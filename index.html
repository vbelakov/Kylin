Kylin Overview

Kylin is an open source Distributed Analytical Engine from eBay Inc. to provide SQL interface and multi-dimensional analysis (OLAP) on Hadoop to support extremely large petabytes of data analysis.

Kylin Ecosystem

    Kylin Core: Fundamental framework of Kylin OLAP Engine comprises of metadata engine, query engine, job engine and storage engine to run the entire stack. It also includes a REST Server to service client requests

    Extensions: Plugins to support for additional functions

    Integration: Lifecycle Management Support to integrate with job scheduler, ETL, monitoring and alerting systems

    User Interface: Allows for third party users to build more features via user-interface atop Kylin core

    Drivers: ODBC and JDBC drivers to support different tools and products, such as Tableau

What is Kylin?

Extremely Fast OLAP Engine at scale:

Kylin is designed to reduce query latency on Hadoop for 10+B rows of data.

ANSI-SQL Interface on Hadoop:

Kylin offers ANSI-SQL on Hadoop and supports most ANSI-SQL query functions.

Interactive Query Capability:

Users can interact with Hadoop data via Kylin Cubes at sub-second latency, better than Hive queries for the same dataset

Pre-Built Data Model:

User can define a data model and pre-build in Kylin with more than 10+ billions of raw data records.

Seamless integration with BI Tools:

Kylin currently offers integration capabilities with Business Intelligence Tools such as Tableau and 3rd party web application. Support for other tools like Microstrategy and Excel is coming soon.

Other Highlights:

  • Job Management and Monitoring
  • Compression and Encoding Support
  • Incremental Refresh of Cubes
  • Leverage HBase Coprocessor for query latency
  • Approximate Query Capability for distinct Count (HyperLogLog)
  • Easy Web interface to manage, build, monitor and query cubes
  • Security capability to set ACL at Cube/Project Level
  • Support LDAP Integration
